# presenter-base

Base para presentador web desarrollado en Sinatra


## Uso

**Rack:**
```
	bundle exec rackup
```

**shotgun:**
```
	bundle exec shotgun config.ru
```


## Dependencias
- [shotgun](https://github.com/rtomayko/shotgun)
- [sinatra](https://github.com/sinatra/sinatra/)
- [thin](https://github.com/macournoyer/thin/)
